# React Forms: Controlled vs Uncontrolled + Fetch (Vite + TS)

## Description
Демо-проєкт для практики:
- Controlled component (форма керується useState)
- Uncontrolled component (форма через useRef)
- Запит до сервера через useEffect + async/await (loading/success/error)

## Technologies
- Vite + React + TypeScript (TSX)

## Install & Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
